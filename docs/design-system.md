# Dizaino sistema

## Kryptis

Svetainė orientuota į žmogų, ypač į būsimą mokinį.

Stilius:
- modernus;
- švarus;
- aiškus;
- neperkrautas dokumentais;
- tinkamas profesinio mokymo centrui.

## Spalvos

Pagrindinė spalva:
- tamsiai mėlyna

Papildomos spalvos:
- šviesiai mėlyna
- plytinė / rusva
- balta
- šviesiai pilka

Vengti:
- žalios kaip pagrindinės svetainės spalvos

Žalia pagrindinio pastato spalva gali būti naudojama tik nuotraukose.

## Tipografika

Naudoti aiškų sans-serif šriftą.

Tekstai turi būti:
- lengvai skaitomi;
- pakankamai dideli;
- aiškiai suskirstyti antraštėmis.

## Mygtukai

Pagrindinis mygtukas:
- tamsiai mėlynas fonas;
- baltas tekstas;
- suapvalinti kampai.

Antrinis mygtukas:
- baltas fonas;
- tamsus tekstas;
- plonas rėmelis.

## Pagrindinio puslapio blokai

1. Header
2. Hero
3. Greitos nuorodos
4. Mokymo programos
5. Padaliniai
6. Naujienos
7. Kodėl rinktis mus?
8. Footer

## Nuotraukos

Hero sekcijoje geriausia naudoti:
- mokinių praktinio mokymo nuotraukas;
- dirbtuves;
- profesinę veiklą;
- gyvą mokyklos aplinką.

Pastatų nuotraukas naudoti padalinių bloke.
