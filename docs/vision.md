# ŠPMC svetainės vizija

## Projekto tikslas

Sukurti modernią, mobiliems įrenginiams pritaikytą Šilutės profesinio mokymo centro svetainę.

Svetainė turi būti orientuota į:

* būsimus mokinius;
* tėvus;
* esamus mokinius;
* socialinius partnerius.

Tuo pačiu turi atitikti visus viešojo sektoriaus informacijos skelbimo reikalavimus.

## Pagrindiniai principai

1. Pirmiausia žmogus, tik po to dokumentai.
2. Modernus ir aiškus dizainas.
3. Greita navigacija.
4. Paprastas turinio administravimas.
5. Mobile-first požiūris.
6. Greitas veikimas.
7. Galimybė plėsti sistemą ateityje.

## Technologijos

Frontend:

* Next.js
* TypeScript
* Tailwind CSS

CMS:

* Strapi

Versijų kontrolė:

* GitHub

Pradinis etapas:

* SQLite
* lokalus kūrimas

Ateities etapas:

* PostgreSQL
* viešas serveris

## Turinys

Svarbiausi akcentai:

* Stojantiesiems
* Mokymo programos
* Naujienos
* Tvarkaraščiai
* Padaliniai

Dokumentai lieka prieinami, tačiau nėra pagrindinis svetainės akcentas.
